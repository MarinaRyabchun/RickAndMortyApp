# RickAndMortyApp
Мини-приложение Рика и Морти со списком персонажей и экраном с подробной информацией о выбранном персонаже.

## Краткое описание:
* Экран со списком персонажей (верстка кодом UIKit)
* Экран с детальной информацией о персонаже (SwiftUI)

## Технологии:

* API:
    - Для персонажей ([API](https://rickandmortyapi.com/api/character))
    - Для локации ([API](https://rickandmortyapi.com/api/location))
    - Для эпизодов ([API](https://rickandmortyapi.com/api/episode))
* Интерфейс:
    - UIKit+SwiftUI
* Архитектура:
    - MVVM

## Требования
* iOS 16.4+
* Xcode 14.3+
* Swift 5,5+

## Установка
* Скачайте репозиторий
* Откройте проект в Xcode
* Запустите проект на Симуляторе
